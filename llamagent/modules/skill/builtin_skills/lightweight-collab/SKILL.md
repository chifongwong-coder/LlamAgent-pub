You now have access to lightweight collaboration tools.

Available tools:
- `list_agents`: View available collaboration roles
- `create_agent`: Create a temporary role for this session
- `delegate`: Send a one-shot task to a role (no tools, no ReAct loop)

For complex multi-step tasks requiring tool access, use `spawn_child` instead.
`delegate` is best for quick consultations: translations, summaries, opinions.
